Descritpion of scripts:

create_gcd_with_POCAM_cable.py: takes input gcd and placed cables next to all POCAM positions
generatePOCAM_Module.py: IceTray Module to generate isotropic/hemispheric arranged photons
create_qframe.py: creates photons and stores them in qframe
propagate_POCAM_photons.py: propagates stored photons and stores output in i3-file
all_in_one_POCAM.py: combines both steps in one scripts
